Informações
Sobre o Reconhecimento de Voz

O sistema usa a Web Speech API do navegador para reconhecimento de voz. Quando o wake word está ativado, o sistema fica sempre ouvindo em background.

Como Funciona
  1. Ative o wake word usando o toggle acima
  2. Diga "ei axis" em qualquer página
  3. O chat do Axis AI abrirá automaticamente
  4. Você pode começar a falar imediatamente
Privacidade

Todo o processamento de voz acontece localmente no seu navegador. Nenhum áudio é enviado para servidores externos.

Nota: O navegador pode solicitar permissão para usar o microfone na primeira vez.
